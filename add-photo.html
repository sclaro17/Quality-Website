QSM Gallery — Add a Photo

The easy way: drop the photo in, double-click one file, push.

Add a photo Recommended

  1. In GitHub Desktop, go to Repository › Show in Explorer. Open the images folder, then the subfolder for the category:
    marquees · in-theatre · custom-displays · conventions · fulfillment

  2. Drop your photo into that folder. The file name becomes the caption, so name it in plain words with dashes instead of spaces:

wicked-for-good-amc-burbank-16.jpg→"Wicked For Good AMC Burbank 16"
f1-imax-amc-boston-common.jpg→"F1 IMAX AMC Boston Common"

Common abbreviations (AMC, CMX, IMAX, XD, RPX, WB…) are capitalized automatically. Stick to letters, numbers and dashes.

  1. Double-click Update Gallery.bat. A small window lists the photos it added and says Done. (If Windows shows a "protected your PC" box, click More info → Run anyway — it's your own file.)

  2. Back in GitHub Desktop: type a summary, click Commit to main, then Push origin.

  3. Open your live site and hard-refresh with Ctrl+F5 to clear the cache. Your new photo is there.

Safe by design: the tool only adds new photos — your existing photos and captions are never changed — and it saves a backup as index.html.bak each run. Removing a photo from a folder and re-running also removes it from the site.
Manual method — only if you ever need it

You normally won't need this. But if the tool isn't available, you can generate one photo's code below and paste it into index.html by hand.

1 · Copy this code
2 · Paste it into index.html

  1. Make sure the photo file is already in the right images subfolder.

  2. Open index.html in a text editor (Notepad, VS Code…).

  3. Press Ctrl+F and find the line for the last photo in your category, then paste your snippet right after that photo's closing </div> — but before the <!--GALLERY-AUTO-END--> line. Save.

3 · Publish it

  1. In GitHub Desktop: write a summary, click Commit to main, then Push origin.

  2. Hard-refresh the live site with Ctrl+F5.

If you later run Update Gallery.bat, it will keep this hand-added photo — no need to redo it.
